# Desk Pet 运行手册

> 版本 1.0.0 | 比赛提交用

---

## 1. 项目简介

Desk Pet 是一款 Windows 桌面 AI 陪伴应用。你可以在桌面上养一个名为"柚子"的 Live2D 角色，与她聊天、管理任务、使用番茄钟、或进入全屏壁纸模式获得沉浸式陪伴。

**三种运行模式：**

| 模式 | 说明 |
|------|------|
| 桌宠模式 | 500×500 透明无边框浮窗，Live2D 角色常驻桌面，支持对话和任务发布 |
| 软件模式 | 1280×800 标准窗口，柚子的房间，任务面板、世界地图、角色互动 |
| 壁纸模式 | 全屏壁纸层，角色大尺寸陪伴，番茄钟+白噪音 |

---

## 2. 快速启动（直接使用 EXE）

`release/` 目录下已提供打包好的可执行文件：

| 文件 | 说明 |
|------|------|
| `DeskPet Setup 1.0.0.exe` | 安装包，支持自定义安装路径 |
| `DeskPet 1.0.0.exe` | 便携版，**双击即用，无需安装** |

推荐直接双击 `DeskPet 1.0.0.exe` 启动。

---

## 3. 配置 API Key

项目使用 DeepSeek API 提供 AI 对话能力。

首次启动后，应用会在同级目录生成 `data/db.json`。用记事本打开，将 `apiKey` 替换为你的 Key：

```json
{
  "appState": {
    "apiKey": "sk-你的DeepSeek-API-Key"
  }
}
```

> 申请地址：https://platform.deepseek.com — 注册后在"API Keys"页面创建

也可以在应用内配置：右键系统托盘图标 → 柚子日记 → 连接页面填写。

## 4. 功能体验指引

### 4.1 桌宠模式（默认启动）

- 桌面出现 Live2D 角色"柚子"，点击她即可对话
- 在对话中表达待办事项（如"下周要考试了"），AI 会自动转化为叙事化任务
- 可以拖动窗口到桌面任意位置
- 右键托盘图标可切换模式、打开设置

### 4.2 软件模式

- 右键托盘图标 → 软件模式（或双击托盘图标）
- 进入"柚子的房间"，可查看任务面板、世界状态、与柚子深度互动

### 4.3 壁纸模式

- 右键托盘图标 → 壁纸模式
- 全屏沉浸式陪伴，内置番茄钟和白噪音

### 4.4 设置（柚子日记）

- 右键托盘图标 → 柚子日记
- 可配置主题、字幕、番茄钟时长等

## 5. 从源码运行（可选）

如果需要从源码启动：

**环境要求**：Node.js ≥ 18、npm ≥ 9、Windows 10/11

```bash
cd desk-pet
npm install
npm start
```

---

## 6. 常见问题

| 问题 | 解决 |
|------|------|
| 启动后 AI 对话无响应 | 检查 `data/db.json` 中 `apiKey` 是否已配置且有效 |
| Live2D 角色不显示 | 确认 GPU 支持 WebGL 2.0 |
| 壁纸模式黑屏 | 重启应用后重新进入壁纸模式 |
| 设置不保存 | 确认 `data/` 目录可写 |
| 杀毒软件报警 | EXE 未签名，信任即可（或从源码运行） |
