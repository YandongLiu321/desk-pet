TypewriterController 类设计

class TypewriterController {
  constructor(speechLabel, options = {}) {
    this.speed = options.speed || 35;       // ms/字 (可配置)
    this.cursorChar = options.cursor || '▍'; // 闪烁光标
    this.writeBuffer = '';    // chunk 累积
    this.displayBuffer = '';  // 当前显示文本
    this.index = 0;
    this.timer = null;
    this.isComplete = false;
  }

  // SSE 每来一个 chunk 就调用
  appendChunk(text) {
    this.writeBuffer += text;
    if (!this.timer) this._startTyping();
  }

  // 定时器逐字输出
  _tick() {
    if (this.index >= this.writeBuffer.length) {
      this._onComplete();
      return;
    }
    this.index++;
    this.displayBuffer = this.writeBuffer.slice(0, this.index);
    this.speechLabel.textContent = this.displayBuffer + this.cursorChar;
    // 每输出一个字 → 触发 Live2D 嘴部参数
    this._triggerLipSync();
  }

  // 流结束后调用
  finalize() {
    this.isComplete = true;  // 全量文本已到达，继续打字直到完成
  }
}

Live2D 嘴部同步

每 tick（输出一个字）调用一次 Live2D 的嘴部参数设置：

  • model.setParameterValueById('ParamMouthOpenY', random(0.3, 0.8))
  • 在每字输出后的 50ms 内触发张嘴，然后回弹到 0
  • 速度可配置：静音模式 vs 有 TTS 时（后续 Phase 2）

气泡行为改进

  • 不再自动消失：打字完成后保持显示，直到下一条消息或用户点击关闭
  • 历史导航：上下箭头按钮浏览之前的回复（借鉴 Sakura 的 reply history）
  • 气泡尺寸：随文本长度自适应，max-height + 滚动
  • 淡入动画：新消息时气泡从角色位置向外微弹出
