Step 1: 主题配色系统

CSS 变量驱动的多主题方案，所有后续 UI 的视觉基础

设计思路

将现有硬编码的暗色星空主题改造为 CSS 自定义属性驱动的主题系统。定义 3 套预设主题，用户可在设置界面切换，选择持久化到 lowdb。

当前状态

variables.css 中变量分散

颜色值直接写在各 CSS 文件中

无切换机制

主题色硬编码在 JS 中

改造后

variables.css — 主题 token 定义 + 3套主题数据属性

base.css — 所有颜色引用 var()

JS ThemeManager — 切换/持久化

IPC — 读写 db 中的 theme 字段

实现要点

  • ThemeManager 类挂在 window.themeManager，所有 renderer 页面共享
  • 切换主题时修改 document.documentElement.dataset.theme，浏览器即时重绘
  • 通过 IPC 读/写 db.data.settings.theme 字段
  • 初始加载时从 db 读取应用上次选择的主题
  • Wallpaper 和 Software 模式的 CSS 变量同步更新
  • 新增 IPC channels: settings:get, settings:set, theme:set
