Step 6: 设置界面

独立设置窗口 + 多标签页 + IPC 读写配置

窗口设计

  • 独立的 BrowserWindow（800x600，可调整大小，非透明）
  • 左侧标签导航（60px 宽图标栏） + 右侧内容区
  • 底部：Save / Cancel / Reset to Defaults 按钮
  • 通过托盘菜单 "Settings..." 或 pet 右键菜单打开
  • 单例模式：重复点击打开同一窗口，focus 而非新建

标签页结构

Tab 1

API 配置

  • API Key (password field)
  • API Base URL
  • Model 选择
  • Test Connection 按钮
Tab 2

外观

  • 主题选择器（3套预设预览）
  • 角色缩放滑块
  • 字幕速度滑块
  • 字体大小
Tab 3

交互

  • 字幕开关
  • 打字机速度
  • 气泡自动消失时间
  • 点击穿透开关
Tab 4

番茄钟

  • 默认时长
  • 短休/长休时长
  • 长休间隔
  • 结束提醒开关

数据流

// 读取设置
Renderer                          Main Process
  │                                  │
  │── ipc.invoke('settings:get') ──→│
  │                                  │── db.data.settings
  │←── { ok, data: settings } ──────│
  │                                  │
// 保存设置
  │── ipc.invoke('settings:set',    │
  │     { theme: 'ocean' }) ──────→ │
  │                                  │── db.update('settings', ...)
  │                                  │── broadcast 'settings:changed'
  │←── { ok, data: null } ───────── │
  │                                  │
// 所有窗口收到 settings:changed → 更新 UI
