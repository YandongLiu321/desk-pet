Step 4: 毛玻璃拟态输入框 & 对话面板

backdrop-filter 玻璃拟态 + 对话面板风格统一

效果目标

输入框和对话面板叠加在 pet 窗口上时，产生景深感——就像一层半透明玻璃悬浮在角色上方，透过它可以隐约看到后面的角色。

当前状态

输入框和面板纯色半透明背景

background: rgba(0,0,0,0.7)

无模糊效果

平面感强，缺少层次

改造后

backdrop-filter: blur(16px) saturate(1.2)

半透明 + 背景模糊 + 微高光边框

对话面板同样应用玻璃拟态

层次分明，现代感

CSS 方案

/* 玻璃拟态基础 mixin（作为 CSS 类复用） */
.glass-panel {
  background: var(--color-bg-glass);
  backdrop-filter: blur(16px) saturate(1.2);
  -webkit-backdrop-filter: blur(16px) saturate(1.2);
  border: 1px solid rgba(255, 255, 255, 0.08);
  border-radius: var(--radius-lg);
  box-shadow:
    0 8px 32px rgba(0, 0, 0, 0.3),
    inset 0 1px 0 rgba(255, 255, 255, 0.05);
}

/* 输入框容器 */
.input-bar {
  composes: glass-panel;
  display: flex;
  align-items: center;
  gap: 8px;
  padding: 8px 12px;
  margin: 12px;
}

/* 输入框本身 - 更轻的玻璃 */
.input-bar input {
  background: rgba(255, 255, 255, 0.06);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: var(--radius-sm);
  color: var(--color-text-primary);
  padding: 6px 10px;
  flex: 1;
  outline: none;
  transition: border-color 0.2s;
}
.input-bar input:focus {
  border-color: var(--color-accent);
  box-shadow: 0 0 0 2px var(--color-accent-glow);
}

/* 对话面板 */
.conversation-panel {
  composes: glass-panel;
  /* ... 同样的玻璃拟态 */
}

应用范围

  • Pet 模式：输入栏 + 对话面板 + 任务面板 + 右键菜单
  • Wallpaper 模式：聊天侧边栏 + 任务选择下拉 + 白噪音控制
  • Software 模式：侧边导航 + 内容卡片 + 输入区域
  • 全部通过 .glass-panel CSS 类复用

注意事项

  • Electron 的 transparent: true 窗口 + backdrop-filter 在 Windows 上需要测试——如果没有背后的像素可采样，模糊可能不生效
  • 备选方案：使用伪元素 + Canvas2D 截取窗口区域做手动模糊（类似 Sakura 的 FrostedGlassFrame），但复杂度高很多
  • 如果 backdrop-filter 不可用，降级为不透明的深色半透明背景
