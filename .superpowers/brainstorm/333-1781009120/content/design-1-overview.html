不打破现有功能的原则

  • IPC 通信保持 { ok, data } / { ok, error } 模式
  • 三模式架构不变（pet/wallpaper/software）
  • WebGL 壁纸引擎不动
  • Live2D 渲染管线不动（只增加嘴部同步参数调用）
  • 番茄钟、任务、关系系统不改
  • lowdb 数据库结构只增字段，不删不改
