# 软件模式 (renderer/software/)

> RPG 风格的游戏化主界面，任务结算与剧情推进。1280×800 独立窗口，暗色星空主题。单文件 `index.html`（纯 HTML/CSS/JS）。

## 子任务

### 页面初始化

- [ ] **sw-01** 创建 `index.html` 单文件页面，三栏布局（左导航 + 中内容 + 右角色区）

### 页面结构

- [ ] **sw-02** 标题栏：深色星空主题，窗口标题 + 模式标签
- [ ] **sw-03** 左侧导航栏：三个标签垂直排列（任务/地图/角色），标签切换通过 JS tab 切换
- [ ] **sw-04** 中间内容区：根据 activeTab 切换显示对应内容
  - 'tasks' → 任务面板（TaskPanel）
  - 'map' → 世界地图（WorldMap 占位）
  - 'character' → 角色档案（背景故事 + 记忆时间线）
- [ ] **sw-05** 任务面板：任务列表 + 选中任务详情 + 子任务勾选（TaskPanel, compact=false）
- [ ] **sw-06** 任务结算：全部子任务完成 → 点击"提交" → IPC `task:complete` → 显示叙事反馈文本 + 动画 + 里程碑进度更新（若章节推进则显示"前往下一章"按钮）
- [ ] **sw-07** 世界地图：原型阶段 CSS 区域占位图 + 区域标记（明亮/灰暗）+ 章节文字。任务结算后根据 `worldState` 更新区域状态（如章节推进后区域变亮）
- [ ] **sw-08** 角色档案：静态背景故事 + 已完成任务列表占位时间线
- [ ] **sw-09** 右侧角色区：CharacterRenderer（mode='software'）+ 底部 ConversationPanel

### IPC 接入

- [ ] **sw-10** 接入 IpcClient：任务列表/详情/子任务勾选/结算
- [ ] **sw-11** 右侧对话面板 IPC：与桌宠共享对话引擎
- [ ] **sw-12** 窗口关闭时返回桌宠：`ipc.switchMode(MODE.PET)`
